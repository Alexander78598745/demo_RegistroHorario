💡 TIPS RÁPIDOS
═══════════════════════════════════════════════════════════════════════════════

🔹 Para pruebas sin esperar 18:00:
   Modifica checkTimeRestriction() en app-admin.js
   Cambia "if (hour >= 18)" por "if (hour >= 0)"

🔹 Para limpiar datos de prueba:
   DevTools (F12) → Application → Local Storage → Clear All

🔹 Para cambiar contraseña de admin:
   Edita app-admin.js → const ADMIN_PASSWORD = 'TuNuevaContraseña'

🔹 Para servidor local de pruebas:
   python -m http.server 8000
